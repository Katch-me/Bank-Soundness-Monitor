## Rule for this project

Never display a metric that isn't in `banks_q3fy26.json`. If a future update adds
a new quarter or a new bank, add it here with its source before adding it to the
JSON. If a figure is genuinely unavailable, show "data pending" in the UI — never
estimate, interpolate, or carry forward an old number silently.
