# RBI Prompt Corrective Action (PCA) Framework — Scheduled Commercial Banks

Source: RBI Circular [RBI/2021-22/118](https://www.rbi.org.in/Scripts/NotificationUser.aspx?Id=12186&Mode=0), effective January 1, 2022.
(Official framework publication: [rbi.org.in](https://www.rbi.org.in) — Master Circular framework for SCBs excluding Small
Finance Banks, Payments Banks and RRBs)

The framework tracks three parameters. A bank is placed in PCA, and faces
escalating restrictions (on dividends, branch expansion, lending to riskier
segments, etc.), the moment it breaches any one of these.

## 1. Capital — CRAR / CET1
Measured as basis points below the regulatory minimum (including the Capital
Conservation Buffer).
- Threshold 1: up to 250 bps below minimum
- Threshold 2: 250–400 bps below minimum
- Threshold 3: more than 400 bps below minimum

## 2. Asset Quality — Net NPA Ratio
- Threshold 1: ≥ 6.0% and < 9.0%
- Threshold 2: ≥ 9.0% and < 12.0%
- Threshold 3: ≥ 12.0%

## 3. Leverage — Tier 1 Leverage Ratio
Measured as basis points below the regulatory minimum.
- Threshold 1: up to 50 bps below minimum
- Threshold 2: 50–100 bps below minimum
- Threshold 3: more than 100 bps below minimum

A bank breaching Threshold 3 on any parameter, or Threshold 2 on two or more
parameters, is generally considered for more severe/mandatory action including
restrictions on management compensation and board-level intervention.

## Important framing note for this project

This tool applies the **public, published methodology** to **public, published
bank data**. It is an independent recreation of RBI's framework for portfolio/
demonstration purposes — it does NOT represent RBI's actual internal supervisory
assessment of any bank, which would also incorporate non-public information,
on-site inspection findings, and supervisory judgment beyond these three ratios.
Say this explicitly in the UI (e.g. a footer disclaimer), not just in this doc.
