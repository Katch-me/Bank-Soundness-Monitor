How a status is assigned

1

Pull published figures

CRAR, CET-1, and Net NPA ratio, taken directly from each bank's quarterly results release.

2

Apply RBI's published thresholds

The same Threshold 1 / 2 / 3 boundaries set out in RBI's PCA circular (RBI/2021-22/118) — nothing simplified or re-defined.

3

Show the distance, not just the verdict

A bank well clear of a threshold and a bank one quarter away from it both read as "Clear" — the headroom number is what actually matters to a reviewer.

What this is, and isn't. This tool applies RBI's publicly available PCA methodology to each bank's publicly disclosed results. It is an independent analytical exercise, not RBI's actual supervisory assessment — which also draws on non-public information, on-site inspection, and supervisory judgment this tool has no access to.
